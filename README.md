# سیستەمی بەڕێوەبردنی مارکێت و کۆگا

سیستەمێکی تەواو بۆ بەڕێوەبردنی فرۆشتن، کۆگا، قەرز و ڕاپۆرتەکان بە PHP و MySQL.

## تایبەتمەندییەکان

- **سیستەمی چوونەژوورەوە** بە چوار ئاستی جیاواز:
  - سوپەر ئەدمین (دەسەڵاتی تەواو)
  - ئەدمین (بێ دەسەڵاتی بەکارهێنەر)
  - کۆگا (تۆمارکردن و سڕینەوەی کاڵا)
  - فرۆشیار (تەنها فرۆشتن)

- **داشبۆردی سەرەکی** لەگەڵ ٣٢ بۆکسی جیاواز:
  - بۆکسەکانی ١-٧: ڕەنگی شین
  - بۆکسەکانی ٨-١٢: ڕەنگی سەوز
  - بۆکسەکانی ١٣-١٧: ڕەنگی پرتەقاڵی
  - بۆکسەکانی ١٨-٢٧: ڕەنگی سور
  - بۆکسەکانی ٢٨-٣٢: ڕەنگی قاوەیی

- **بانەری سەرەوە**ی ٣٠px کە ناوی بەکارهێنەر و ڕێکەوت پیشان دەدات

- **سیستەمی فرۆشتن** لەگەڵ سەبەتەی کڕین و باڕکۆد

- **بەڕێوەبردنی کاڵا** زیادکردن، دەستکاریکردن و سڕینەوە

- **سیستەمی قەرز و قیست** تۆمارکردن و وەرگرتنەوە

- **ڕاپۆرتەکان** ڕاپۆرتی فرۆشتن، قازانج و کەشف حساب

## پێویستییەکان

- PHP 7.4 یان زیاتر
- MySQL 5.7 یان زیاتر
- Web Server (Apache/Nginx)
